# RFV Vorderrhoen Website

Static Astro marketing site for an equestrian vaulting club, built for GitHub Pages deployment.

## Stack

- Astro
- TypeScript
- Tailwind CSS
- React islands
- Framer Motion
- Astro Content Collections

## Run locally

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```
